GWeb  
====

GWeb a is hybrid Web application to run jobs for genomic analysis using [Genestrip](https://github.com/pfeiferd/genestrip).

## License

[GWeb is free for non-commercial use.](./LICENSE.txt) Please contact [daniel.pfeifer@progotec.de](mailto:daniel.pfeifer@progotec.de) if you are interested in a commercial license.

## Building and installing

GWeb is structured as a standard [Maven 2 or 3](https://maven.apache.org/) project with modules and is compatible with the [JDK](https://jdk.java.net/) 1.8 or higher.
Generating the a Windows executable of GWeb requires the JDK 11 or higher on a Windows machine as it uses [JLink](https://docs.oracle.com/en/java/javase/11/tools/jlink.html).
